# Progetto_Logistica
Il progetto è inerente all' esame di Laboratorio di Informatica (corso di laurea triennale in ingegneria elettronica Unical) .Si tratta di un Hub di spedizioni centralizzato. Il programma gestisce le interazioni tra l'azienda logistica e i suoi clienti, i quali operano tramite un sistema a portafoglio prepagato per inviare e ricevere pacchi.
